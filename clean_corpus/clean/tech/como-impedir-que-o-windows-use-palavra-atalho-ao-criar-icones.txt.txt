Ao criar um novo atalho no Windows, automaticamente, é adicionada a palavra “atalho” no nome do arquivo criado.
Isso é feito para diferenciar o que é um arquivo normal do que é um atalho.
Além dessa diferença, existe uma seta de atalho sobreposta aos seus ícones.
Caso o usuário queira impedir que o Windows adicione a palavra ''atalho'' ao criar seus respectivos ícone, basta seguir os passos desse tutorial.
Como criar um código PIN complexo e mais seguro no Windows 10 Devido ao risco dessas modificações, antes de começar tenha certeza do que está fazendo e crie um ponto de restauração do sistema.
Vale lembrar que é possível reverter mudanças no registro do Windows 10.
Aplicativo do Tech Tudo: receba as melhores dicas e últimas notícias no seu celular Passo 1.
Execute o Editor de Registro do Windows, pressionando as teclas ''Windows + R'' simultaneamente.
Na pequena janela que aparece, digite “regedit” (sem as aspas) e tecle Enter.
Se for solicitado, forneça a senha ou clique no botão “Sim” para autorizar a execução do programa; Passo 2.
Com o Editor do Registro aberto, navegue até a chave “HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\Current Version\Explorer”; Passo 3.
Com a chave “Explorer” selecionada, clique com o botão direito do mouse em uma área vazia do quadro direito.
No menu exibido, clique em “Novo” e depois em “Valor Binário”; Passo 4.
Assim que o valor for criado, coloque a palavra “link” no lugar do nome padrão e tecle Enter para confirmar; Passo 5.
Dê um clique duplo no novo valor para poder editá-lo.
Quando a janela de edição aparecer, mude o valor do campo “Dados do valor:” para “00 00 00 00″ e depois confirme, clicando no botão “OK”; Pronto!
Você terá de reiniciar o computador (ou terminar a sessão e voltar a entrar) para concluir a alteração.
Teste a mudança criando um novo atalho.
Se você quiser reverter as mudanças, basta voltar para o registro e excluir o valor do item ''link''.
Isso fará com que o Windows recrie o valor com a configuração padrão.
