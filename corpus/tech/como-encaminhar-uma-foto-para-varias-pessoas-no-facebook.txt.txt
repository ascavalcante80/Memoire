O Facebook permite ao usuário encaminhar uma imagem recebida no bate-papo para diversos amigos. A função, que é ideal para apresentar uma foto com um grupo ou uma arte de eventos para muitas pessoas da lista de contatos, envia o arquivo em um chat de mensagem individual. Sendo assim, o encaminhamento respeita a privacidade das pessoas envolvidas, pois não é criado um grupo de conversa para que a imagem seja visualizada. Para saber como usar o encaminhamento de imagens para vários contatos do Facebook, confira o tutorial.
